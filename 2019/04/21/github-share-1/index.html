github项目分享一

yapi

  • 名称：yapi
  • Github地址：https://github.com/ymfe/yapi
  • 官方教程：https://yapi.ymfe.org/documents/index.html
  • 语言：nodejs
  • 定位：YApi 是一个可本地部署的、打通前后端及QA的、可视化的接口管理平台
  • 关键词：api管理平台 、api文档 、mock
  • 特性：
    1. 基于 Json5 和 Mockjs 定义接口返回数据的结构和文档
    2. 扁平化权限设计，即保证了大型企业级项目的管理，又保证了易用性
    3. 类似 postman 的接口调试
    4. 自动化测试, 支持对 Response 断言
    5. 支持 postman, har, swagger 数据导入
    6. 免费开源，内网部署

canal

  • 名称：canal [kə’næl]
  • 地址：https://github.com/alibaba/canal
  • go客户端地址：https://github.com/CanalClient/canal-go
  • 官方文档：https://github.com/alibaba/canal/wiki/AdminGuide
  • 语言：纯java开发
  • 定位：基于数据库增量日志解析，提供增量数据订阅&消费，目前主要支持了mysql
  • 关键词：mysql、binlog parser、real-time、queue&topic
  • 基于日志增量订阅&消费支持的业务：
    1. 数据库镜像&备份
    2. 异构数据库同步
    3. 多级索引 (卖家和买家各自分库索引)
    4. 搜索引擎增量 build（elatiseach or sorl）
    5. 业务cache刷新
    6. 价格变化等重要业务变更消息

SOAR

  • 名称：SOAR
  • 地址：https://github.com/XiaoMi/soar
  • 语言：go语言开发
  • 定位：对SQL进行优化和改写的自动化工具，目前主要支持了mysql
  • 关键词：sql parser、sql optimization、sql rewriting
  • 功能特点：
    1. 跨平台支持
    2. 目前只支持 MySQL 语法族协议的SQL优化
    3. 支持基于启发式算法的语句优化
    4. 支持复杂查询的多列索引优化（UPDATE, INSERT, DELETE, SELECT
    5. 支持EXPLAIN信息丰富解读
    6. 支持SQL指纹、压缩和美化
    7. 支持同一张表多条ALTER请求合并
    8. 支持自定义规则的SQL改写

Supervisor

  • 名称：Supervisor
  • 地址：https://github.com/Supervisor/supervisor
  • 语言：python
  • 定位： Supervisor是一个Python开发的client/server系统，可以管理和监控类unix上面的进程
  • 关键词：进程，拉起，管理
  • 功能特点：
    1. 启动、重启、关闭进程
    2. 服务器出问题导致进程都被杀死，用 supervisor 自动重启
  • 教程参考：Supervisor简介、安装、配置

etcd

  • 名称：etcd
  • 地址：https://github.com/etcd-io/etcd
  • 语言：go语言开发
  • 定位：高可用的分布式键值(key-value)数据库。etcd内部采用raft协议作为一致性算法
  • 关键词：raft算法 | 服务注册 | 服务发现 | 分布式锁
  • 功能特点：
    1. 简单：安装配置简单，而且提供了HTTP API进行交互，使用也很简单
    2. 安全：支持SSL证书验证
    3. 快速：根据官方提供的benchmark数据，单实例支持每秒2k+读操作
    4. 可靠：采用raft算法，实现分布式系统数据的可用性和一致性
  • 应用场景
    • 一个强一致性、高可用的服务存储目录
    • 一种注册服务和健康服务健康状况的机制
    • 一种查找和连接服务的机制
