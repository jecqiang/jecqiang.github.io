开发工具分享

前言

作为一个Mac重度用户兼程序猿，决定分享一波自己的开发工具清单。希望这些开发工具能够提高小伙伴们的开发工作效率。

推荐工具

1. iTerm2
推荐理由： macOS终极终端，我用过的最好的终端。功能包括：拆分窗格、热键窗口、搜索、自动完成、粘贴历史记录、即时重播、全局热键等等等。
下载链接： https://www.iterm2.com/downloads.html

2. zsh
推荐理由：系统的默认 Shell 都是 bash，但是真正深藏不露 Shell 是 zsh , 堪称『终极Shell』
Github地址：https://github.com/robbyrussell/oh-my-zsh/wiki/Installing-ZSH

3. oh my zsh
推荐理由：oh my zsh是让你快速上手zsh项目。让你对zsh爱不释手。
Github网址： https://github.com/robbyrussell/oh-my-zsh
主题配置：https://github.com/robbyrussell/oh-my-zsh/wiki/Themes
插件推荐：git（默认）、osx、autojump（智能跳转，墙裂推荐）

3. Dash
推荐理由：开发者必备的API文档下载和浏览工具，有了Dash，再也不用到处找API文档了，内置了丰富的API文档，可以在线下载各种开发API和文档资料
下载地址：https://kapeli.com/dash

4. Navicat
推荐理由：所用过最优秀的数据库管理客户端，支持MySQL、SQL Server、SQLite、Oracle 和 PostgreSQL 等数据库，支持数据库建模，正向和反向工程，数据同步等
下载地址：https://www.navicat.com/en/company/press/107-pres

5. Postman
推荐理由：很好用的一款API/接口功能测试工具，很方便的塞数据，查看响应，设置检查点/断言，能进行一定程度上的自动化测试
下载地址：https://www.getpostman.com/
详细教程：https://www.jianshu.com/p/6c9b45994c34

6. Charles
推荐理由：中文名叫青花瓷，web代理，HTTP监听。Charles是一款抓包神器，基于 java 开发的。
下载地址：https://www.charlesproxy.com/download/latest-release/
详细教程：http://blog.devtang.com/2015/11/14/charles-introduction/

7. alfred
推荐理由：Mac下大名鼎鼎的效率神器—Alfred，Mac「装机必备」，只有你想不到，没有它做不到。搜索、启动app、计算、标签……
下载地址：https://www.alfredapp.com/
详细教程：https://www.jianshu.com/p/cf16b2c973e9

8.Docker
推荐理由：必须要掌握docker容器
下载地址：https://hub.docker.com/editions/community/docker-ce-desktop-mac

9. Balsamiq Mockups
推荐理由：手绘版原型图绘制工具，简单易用好看
下载地址：https://balsamiq.com/wireframes/

10. redis-stat
推荐理由：简单轻量的redis监控工具
Github地址：https://github.com/junegunn/redis-stat

11. mycli
推荐理由：支持高亮、命令自动提示+补全的Mysql命令行工具
下载地址：https://www.mycli.net/

12. PhpStorm
推荐理由：个人觉得PHP开发最好用了IDE
下载地址：https://www.jetbrains.com/phpstorm/download/

13. LightPaper
推荐理由：Markdown编辑器，轻量化的同时也配合上了实用的功能。这边博文也是用改软件来写的
下载地址：https://getlightpaper.com/

14. iPic
推荐理由： 图床神器 iPic，写博客、在公众号发文章、在知乎讨论、在豆瓣灌水、在论坛发帖都能用得上，偷偷告诉你该博文图片也是iPic上传的
下载地址：https://toolinbox.net/iPic/
